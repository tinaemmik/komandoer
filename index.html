Kommandoguide

Git clone <url>

Git clone er den kommando man bruger i terminalen for at hente den online version af filerne ned på ens computer. Dette kan KUN gøres første gang man henter filerne ned.

Git status

Viser status over filerne

Git add .

Fortæller git, at der er tilføjet ændringer.

git commit -m "message"

Her kan man sende en besked med sin opdatering på den nye version

git push

Denne bruges når man skal skubbe de lokale filer ind i GitHub
